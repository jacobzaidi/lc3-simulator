Part1.c converts each line of an assembly program into hexadecimal form.

Part2.c  implements an LC-3 simulator that is capable of executing the following instructions:
 * LD
 * LEA
 * LDI
 * AND
 * NOT
 * ADD

The following assembly programs are provided within the simulator for testing:
 * p1.obj
 * p2.obj
 * p3.obj
 * p4.obj
 * p5.obj
 * p6.obj
 * p7.obj
